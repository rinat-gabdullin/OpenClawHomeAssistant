# NocoDB Add-on

[NocoDB](https://nocodb.com) — open source альтернатива Airtable. Превращает любую базу данных в умную совместную таблицу с богатым веб-интерфейсом, REST/GraphQL API и поддержкой нескольких баз данных.

## Запуск

После установки и запуска add-on откройте вкладку **Web UI** или перейдите по адресу `http://<HA_IP>:8080`.

При первом входе создайте учётную запись администратора через интерфейс NocoDB.

## Настройки

| Параметр | Описание | По умолчанию |
|---|---|---|
| `timezone` | Временная зона | `Europe/Sofia` |
| `port` | Порт веб-интерфейса NocoDB | `8080` |
| `database_url` | URL внешней БД (если пусто — используется встроенный SQLite) | — |
| `jwt_secret` | JWT-секрет для аутентификации (если пусто — генерируется автоматически) | — |
| `public_url` | Публичный URL NocoDB (для корректной генерации ссылок) | — |

## Поддерживаемые базы данных

По умолчанию используется встроенный **SQLite** — данные хранятся в `/config/.nocodb/noco.db`.

Для подключения внешней базы данных задайте `database_url`:

| Тип | Формат |
|---|---|
| PostgreSQL | `pg://user:pass@host:5432/db` |
| MySQL | `mysql2://user:pass@host:3306/db` |

## Хранение данных

Все данные (база SQLite, загрузки, плагины, JWT-секрет) хранятся в `/config/.nocodb` — **переживают обновления add-on'а**.

## API

NocoDB автоматически предоставляет REST и GraphQL API для всех таблиц. Документация API доступна по адресу `http://<HA_IP>:8080/swagger`.
